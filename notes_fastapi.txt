we need Fastapi and a webserver(for python it is uvicorn)

################################
The Frontend (React/Vue) runs in the user's browser.
    |
When the user types a question, the Frontend sends an HTTP request to Uvicorn.
    |
Uvicorn passes that request to FastAPI.
    |
FastAPI runs your BasicChain, gets the answer, and sends it back through Uvicorn to the Frontend. 
##################################
